# meow

A playful and fun way to display messages in the console. With meow, you can
print your custom text alongside a cute ASCII cat.

## Usage

```typescript
import { meow } from "@katayama8000/meow";

meow("Hello, world!");
```

```bash
 |\_/|     
 (. .)   Hello, world!
  =w= (\   
 / ^ \//

,""_""_ .
```
